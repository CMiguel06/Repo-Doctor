# Security Policy

Repo Doctor is a local repository health analyser. It may detect paths, filenames, configuration files, tokens, keys and other sensitive indicators.

## Security principles

- Run only on repositories you own or are authorised to inspect.
- Treat generated reports as potentially sensitive.
- Do not publish reports without reviewing findings.
- Do not use Repo Doctor to inspect private repositories without permission.
- The tool does not upload files or reports to external services.

## Reporting vulnerabilities

If you identify a vulnerability in Repo Doctor, report it privately to the maintainer.
